# **App Name**: Highlander Tracker

## Core Features:

- Trip List View: Display a list of available trips from Mantalongon to Cebu City, filtering by bus type (Traditional or Airconditioned).
- Interactive Seat Map: Visualize the seat map for a selected trip, showing available, selected, and reserved seats. Support different layouts based on bus type.
- PDF Receipt Download: Allow users to download a PDF receipt of their reservation with details such as passenger name, seat number(s), bus type, and departure time.
- Real-Time Bus Tracking: Display real-time GPS tracking of buses on a map as they travel from Mantalongon to Cebu City, updating the bus location frequently.
- Smart Departure Time Suggestions: Use a generative AI tool to suggest optimal departure times, based on an analysis of typical traffic conditions at different times of the day, using the origin and destination as tool inputs.
- Login and Signup: Allow users to create an account and log in to the application.

## Style Guidelines:

- Primary color: Orange (#FF6600) to convey energy and excitement for travel.
- Background color: Black (#000000), providing a sleek and modern backdrop.
- Accent color: Metallic Gray (#A9A9A9) to highlight interactive elements and call-to-action buttons, giving a sense of sophistication.
- Use clear and readable fonts such as 'Roboto' or 'Open Sans' for optimal readability on all screen sizes.
- Employ minimalist icons to represent bus types, seat availability, and map markers.
- Design a clean and intuitive layout with clear visual hierarchy for easy navigation and user experience.
